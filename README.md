# Dashboard Comercial y Ventas - Northwind

## Objetivo
Desarrollar una solución de Business Intelligence para analizar ventas, categorías de productos, regiones y tipos de envío.

## Herramientas Utilizadas
- Power BI
- SQL
- Excel

## Funcionalidades
- KPIs de ventas
- Análisis por categoría
- Análisis por región
- Segmentadores dinámicos
- Visualizaciones interactivas
